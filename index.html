Tecnologias usadas no Desenvolvimento Front-End

GitHub Linkedin

HTML

#001
Linguagem de Marcação

Descrição

Define a estrutura básica das páginas web. É usado para criar a base de qualquer site, organizando títulos, parágrafos, imagens, links e formulários. Sem HTML, não haveria uma base para o conteúdo.

CSS

#002
Linguagem de Estilo

Descrição

Estiliza o conteúdo criado com HTML, controlando o layout, cores, fontes e espaçamentos. CSS também permite a criação de designs responsivos que se adaptam a diferentes tamanhos de tela.

JavaScript

#003
Linguagem de Programação

Descrição

Traz interatividade para o front-end. Usado para manipular o DOM, validar formulários, criar animações e carregar dados dinamicamente, permitindo uma experiência de usuário mais rica e envolvente.

TypeScript

#004
Linguagem de Programação

Descrição

Um superconjunto do JavaScript que adiciona tipagem estática. Ele ajuda a evitar erros comuns e torna o código mais robusto e fácil de manter, especialmente em grandes projetos.

React

#005
Biblioteca JavaScript

Descrição

Uma biblioteca JavaScript para construir interfaces de usuário baseadas em componentes. React facilita a criação de interfaces dinâmicas e interativas, permitindo a reutilização de código em diferentes partes do projeto.

Bootstrap

#006
Framework

Descrição

Um framework CSS popular para criar layouts responsivos rapidamente. Ele fornece componentes pré-construídos, como botões, grids e modais, agilizando o desenvolvimento sem a necessidade de escrever CSS personalizado.

Tailwind CSS

#007
Framework

Descrição

Um framework utilitário que permite criar designs diretamente no HTML usando classes pré-definidas. É altamente customizável e oferece flexibilidade sem depender de CSS escrito à mão.

Vue

#008
Framework

Descrição

Um framework JavaScript progressivo, simples e flexível, usado para criar interfaces de usuário. É fácil de integrar com projetos existentes e oferece um ecossistema robusto para criar aplicações modernas.

Angular

#009
Framework

Descrição

Um framework de front-end completo, desenvolvido pelo Google, que facilita a criação de aplicações escaláveis e complexas, com uma abordagem estruturada baseada em componentes e injeção de dependência.

Jest

#010
Framework

Descrição

Um framework de testes JavaScript, focado em testes unitários e de integração, especialmente para aplicações React. Ele é fácil de configurar e oferece relatórios detalhados de erros e cobertura de código.

Next.js

#011
Framework

Descrição

Um framework para React que oferece renderização no servidor e geração de sites estáticos. Ele melhora o SEO e o desempenho, com recursos como rotas dinâmicas e otimização automática.

Sass

#012
Pré-processador

Descrição

Um pré-processador CSS que permite o uso de variáveis, aninhamentos e funções no código, tornando-o mais modular e fácil de manter, especialmente em projetos grandes.

Vite

#013
Ferramenta de Build

Descrição

Uma ferramenta de build moderna para desenvolvimento front-end. Usa um servidor de desenvolvimento rápido e compila o código de forma otimizada para produção, melhorando a eficiência no fluxo de trabalho.

Cypress

#014
Ferramenta de Teste

Descrição

Uma ferramenta de teste end-to-end que simula o comportamento do usuário para testar toda a aplicação. É excelente para garantir que todas as interações do front-end funcionem como esperado.

API Rest

#015
API

Descrição

Um estilo arquitetônico que permite a comunicação entre o front-end e o back-end através de métodos HTTP. Muito usado para acessar e manipular dados de servidores remotos.

GraphQL

#016
Linguagem de Consulta APIs

Descrição

Uma linguagem de consulta para APIs que permite buscar exatamente os dados necessários. É mais eficiente que REST em certos cenários, pois reduz o número de chamadas de rede e a quantidade de dados retornados.
